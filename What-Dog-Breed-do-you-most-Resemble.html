Project Definition

Project Overview

Image classification is becoming more and more prominent in today’s evolving world. From security imaging software, to something as simple as unlocking your phone, image recognition algorithms have become integral in our lives. The growth of this technology really blossomed thanks to the convolutional neural networks (or CNNs).

In this project I will try and create an image classifier of my own using CNNs. The algorithm will first look at a picture that I give it and then determine if it is a human, a dog, or neither. If the image is a dog, the classifier will then determine what breed the dog is. If the image is a human, then the classifier will determine what breed the human most closely resembles. If the image is neither human nor dog, the image classifier will opt not to classify the image.

Problem Statement

The project workflow is as follows:

  1. Download and preprocess training datasets of images for humans and dogs
  2. Create a classifier that can detect a human in an image
  3. Create a classifier than can detect a dog in an image
  4. Create CNN to a classify dog breed and train it on training dataset for dogs
  5. Create algorithm that does the following:
    a. Classify the image as a dog, human, or neither
    b. If human, predict most similar dog breed
    c. If dog, predict dog breed
    d. If neither, do not classify the image

Metric

For rating our algorithm we will be concerned with how well our classifier is working. Thankfully in our data, it is known if the image is a human, a dog, and what breed each dog is. This makes our very easy to calculate, as we can simply calculate how many correctly classified images we have by comparing our predictions against the correct results.

Analysis

Data Exploration

Much of the data I would use to train the classifier had been given to me at the start of the project. The human dataset had 13233 total human images, while the dog dataset had 8351 total dog images. In the dog dataset there were 133 different dog breeds. A simple investigation into the data shows us that all of the human images a clear image and the person is directly facing the camera, which is something we should be weary about.

This limits our human detection algorithm as a person who’s face is partly covered or who’s head is tilted may not be recognized by this algorithm. We will analyze this in more detail later.

Methodology

Preprocessing Data

As you may have inferred from above, each of our images need to have a certain input shape as a 4-D vector in order for the algorithm to work. A summary of the process is as follows

  1. Loaded each image with a shape of 224x224 pixels
  2. Converted this image to a 3-D vector using keras img_to_array function (The 3 dimensions are for R,G, and B)
  3. Expanded the single image to a 4-D tensor so the final shape of the array was (1, 224, 224, 3). This could then be fed into the CNN above

Algorithms and Techniques

The dog breed classifier that I wrote uses CNNs and is a very common method used for classification. Before that however, there were two other classification methods that were used; remember we first had to determine if our image was a human, a dog, or neither. Thus I imported two separate classifiers to do this.

The first used ResNet-50 model from Keras, which essentially contains millions of images for many different objects. I used this model to predict whether our image contained a dog or not. Additionally I used OpenCV’s implementation of Haar feature-based cascade classifiers to detect human faces in images. I won’t spend any more time talking about these two as you can look up documentation on this elsewhere.

Now the fun stuff..

The classifier that I chose to create contained 3 convolutional layers, each followed by max pooling layers to keep the dimensionality low. The 3 convolutional layers allow the classifier to learn about the higher level features in the image while the 3 max pooling layers keeps the dimensionality low enough to run in an acceptable amount of time. Finally I added a dense to sort the dog breeds into 1 of the 133 different dog breeds that we know exist in the set. We can then compare this to the actual dog breeds given to determine how accurate our algorithm is.

I would find through trial and error that I could improve the above model using transfer learning.

Refinement

Actually training the model a few trials and errors. I first tried to use the classifier as shown in the algorithms and techniques section, but realized that I could achieve a much higher score if I instead used a prexisting classifier from ResNet50 that has already been train on many more images than our small sample size had. Thus, I eventually settled on training a simpler CNN using transfer learning from ResNet50’s trained model. Thus the final model I settled on had a summary as shown below:

Essentially the input to this model would be the output from the trained ResNet50 model, that would be much more accurate than my model as it had a much larger sample size. Then I could condense the results down into the 133 categories as before, which represent our 133 different dog breeds. This model would train quicker and be more accurate than our initial model.

From here I fit the model using the ResNet50 data and cross-checking it against out training data. I set the batch size and number of epochs to 20 when fitting.

Results

Evaluation and Validation

After training the data using transfer-learning I achieved an accuracy of 80.4%! This is a huge step up from randomly guessing a dogs breed, which would have statistically given us an average of <1% (1/133).

Justification

Dog Classifier
The dog classifier worked the best of any of the classifiers, likely because it was trained using the ResNet50 model. despite this there were still areas where it had trouble, mainly with images of dogs that were not zoomed in on the dogs face. Side profiles seemed to mess with the algorithm and could likely have been fixed by adding different orientations to the sample data.

Human Classifier
By far the weakest part of my classifier. Due to the nature of the training data, this classifier would only work on clear un-obstructed facial images. As I pointed out earlier this would likely lead to problems down the road, and it did indeed.

Conclusion & Moving Forward

Reflection

The goal of this project was to create an image classifier. The algorithm would first detect if the image was a dog, human, or neither. If the image was a dog then the algorithm would predict the breed, if the image was a human it would predict a breed the person most closely resembles, and if the image was neither a dog or a human the algorithm should not classify the image. I ultimately settled on using a CNN created using the ResNet50 trained model that achieved an accuracy of 80.4%.

Moving Forward

There are a few things I think should be done moving forward:

  1. Use a different face detector. Preferably one that was trained on images where the face is partly covered, turned, and at different angles. The face classifier we are using now is too specific on the input image to work on a broader scale.
  2. Try using a different model for transfer-learning at the beginning of our CNN. We used ResNet50, we could have used VGG19, Inception, Xception, and likely many more.
  3. Add more layers in between initial Resnet50 input layer to the final output layer that contains the 133 dog breeds. As my CNN experience is not very large, I don’t know what would be best to do here, but this algorithm can definitely be improves with at least one additional layer
